# num2rus v 0.0.11

Небольшой python пакет для преобразования чисела в текст с учетмо множественного числа. 
A small package to convert float numbers to russian string.

## What's new
- bug fix for #8 issue

## Usage

### Install package

`pip install num2rus`

### Import it

```python
 import num2rus
 num2rus.converter(1.10) # один рубль десять копеек
```
